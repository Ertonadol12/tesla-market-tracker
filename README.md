# 🚗 Tesla Used Inventory Tracker & Cross-Brand Market Comparator

## Overview
Automated platform that scrapes Tesla and competitor used inventory to provide market intelligence.

## Features
- ✅ Tesla API integration
- ✅ Ford, BMW, Hyundai, Kia, Toyota, Mercedes, VW, Polestar tracking
- ✅ Daily automated updates
- ✅ Interactive Streamlit dashboard
- ✅ Depreciation analysis
- ✅ Price per mile efficiency
- ✅ Geographic price variation

## Installation

```bash
git clone https://github.com/Ertonadol12/tesla-market-tracker.git
cd tesla-market-tracker
pip install -r requirements.txt
playwright install
streamlit run dashboard/app.py
